# Strings

DaisyDuino patch based on the Karplus-Strong plucked string model.

https://electro-smith.github.io/DaisySP/classdaisysp_1_1_string_voice.html

## Knobs
- CV_1 - tune
- CV_2 - brightness
- CV_3 - damp
- CV_4 - non-linearity

## Inputs
- B10 - gate in
- CV_5 - volt/oct
- CV_6 - brightness
- CV_7 - damp
- CV_8 - non-linearity

## Outputs
- OUT_L and OUT_R - stereo output
- CV_OUT_1 - lfo out
